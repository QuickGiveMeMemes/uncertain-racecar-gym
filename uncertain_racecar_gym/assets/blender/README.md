# Blender Replay Notes

Use the replay bundle generated by `uncertain-racecar-export-replay` to drive an offline scene in Blender.

Recommended pipeline:
1. Build or import the track mesh from the centerline CSV referenced in the bundle.
2. Import the simple racecar asset or replace it with a higher-fidelity model.
3. Animate the vehicle from `trajectory.json`.
4. Apply the camera cuts from `camera_script.json`.
